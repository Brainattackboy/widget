Curse Download Widget

Widgets for curse.com hosted projects, quick and easy project overview and download information anywhere on the web.

Summary of what this is, mention about the json API for anyone wanting to create their own system, also include a mention of the open source nature of the project (link here) and encourage people to contribute their own widget designs if they want to.

Minecraft Forum

Embedding a curse.com project widget on minecraftforum.net is simple, use the mod or texturepack BBCodes to place any number of widgets anywhere in your forum topics. Project identifiers can be found via the project's curse.com URL, optionally include a version number to specify which file will be downloaded.

Mod

Embed a download widget for the latest version of Mine & Blade: Battlegear 2

[mod]mb-battlegear-2[/mod]

Embed a download widget for version 1.6.4 of Mine & Blade: Battlegear 2

[mod="1.6.4"]mb-battlegear-2[/mod]

Texture Pack

Embed a download widget for the latest version of Equanimity [32x]

[texturepack]equanimity-32x-1-7-x-1-6-x[/texturepack]

Embed a download widget for version 1.7.2 of Equanimity [32x]

[texturepack="1.7.2"]equanimity-32x-1-7-x-1-6-x[/texturepack]

Elsewhere

Widgets are embedded using iframes, create an iframe anywhere on your page referencing widget.mcf.li followed by the curse.com identifier for your project. The identifier of your project is the path immediately following curse.com, for example the identifier for curse.com/mc-mods/minecraft/mb-battlegear-2 is mc-mods/minecraft/mb-battlegear-2.

API

Append .json to any widget URL and a full JSON representation of all of the data available for a project will be returned, eg: mc-mods/minecraft/mb-battlegear-2.json

Notes

Project data is cached for an hour.
