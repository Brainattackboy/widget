Curse Download Widget

Embeddable widgets for curse.com hosted projects, share project information and downloads anywhere on the web, fast and efficiently.

Share your mod, map and texture pack on the Minecraft Forum using the BBCode, access your project meta data using the JSON API, embed widgets on your own website using iframes and contribute design and development improvements to the widget on GitHub.

Minecraft Forum

Embedding a curse.com project widget on minecraftforum.net is simple, use the mod, texturepack or world bbcode to place any number of widgets anywhere in your forum posts. Projects are identified by their curse.com slug or a unique CurseForge project ID.

The optional version parameter accepts a Release Type, Major Version, Major Version with Release Type, or File ID.
