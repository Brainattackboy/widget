Curse Download Widget

Embeddable widgets for curse.com hosted projects. Share project information and downloads anywhere on the web, fast and efficiently.

Share your mod, map and texture pack on the Minecraft Forum using the BBCode, access your project data using the JSON API, embed widgets on your own website using iframes and contribute design and development improvements to the widget on GitHub.

Minecraft Forum

Embedding a curse.com project widget on minecraftforum.net is simple, use the mod, texturepack or world bbcode to place any number of widgets anywhere in your forum topics. Projects are identified by their curse.com slug or a unique CurseForge project ID.

The optional version parameter accepts a Release Type, Major Version, Major Version with Release Type, or File ID.

Mod

Embed a download widget for the latest version of Mine & Blade: Battlegear 2

[mod]mb-battlegear-2[/mod]

Embed a download widget for version 1.6.4 of Mine & Blade: Battlegear 2

[mod="1.6.4"]mb-battlegear-2[/mod]

Embed a download widget for the latest file with a release type beta of Mine & Blade: Battlegear 2

[mod="beta"]mb-battlegear-2[/mod]

Embed a download widget for the version 1.6.2 beta of Mine & Blade: Battlegear 2

[mod="1.6.2/beta"]mb-battlegear-2[/mod]

Embed a download widget for the #727805 file of Mine & Blade: Battlegear 2

[mod="727805"]mb-battlegear-2[/mod]

Elsewhere

Widgets are embedded using iframes, create an iframe anywhere on your page referencing widget.mcf.li followed by the curse.com identifier for your project. The identifier of your project is the path immediately following curse.com, for example the identifier for curse.com/mc-mods/minecraft/mb-battlegear-2 is mc-mods/minecraft/mb-battlegear-2.

<iframe src="http://widget.mcf.li/mc-mods/minecraft/mb-battlegear-2" width="100%" style="border: none;"></iframe>

Optionally specify the version to be downloaded using the version parameter, by default the latest version will be used.

<iframe src="http://widget.mcf.li/mc-mods/minecraft/mb-battlegear-2?version=1.6.4" width="100%" style="border: none;"></iframe>

API

Append .json to any widget URL and a full JSON representation of all of the data available for a project will be returned, eg: mc-mods/minecraft/mb-battlegear-2.json

Notes

Project data is cached for an hour.
